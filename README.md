\# Faithful Reboot



A reverse engineering and preservation project for Trade Nations.



\## Structure



\### original/

Contains extracted original game files:

\- TradeNations.app

\- Original IPA archive

\- Payload extraction



\### research/

Contains extracted text, strings, and notes.



\### tools/

Contains scripts used for extraction and analysis.



\## Goal



Reconstruct the gameplay systems, assets, and mechanics of Trade Nations using modern tools.
