# assets/audio/

Reserved for Phase 5 (`docs/IMPLEMENTATION_ROADMAP.md`). No audio has been
added yet.

The original ships audio as `.caf` (Apple Core Audio Format) files indexed
by `Sounds.xml` (id → filename/volume/isMusic). Converted `.ogg`/`.wav`
versions go here once a Sounds table is converted (Phase 1.3) and something
in Phase 2+ needs to play them.
